This plugin demonstrates how to upload a csv file WordPress and process it.

== Description ==

This plugin demonstrates how to upload a file in the admin side of WordPress 
without the media library interface. It also processes it and imports categories.
You can use this plugin as a base for doing other tasks. 
You should combine this with the batch example if you are going to do something timeconsuming.
