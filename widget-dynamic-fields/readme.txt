=== Widget Drop Down Demo ===

Contributors: flacoman91
Tags: example, plugin development
Requires at least: 3.0.1

This plugin demonstrates how to simulate creating dynamic fields in a widget.

== Description ==

Typically, if you have a ton of fields in a widget, it can end up looking cluttered when you first open it up.
With this code, you can simulate making dynamic fields, and only "add" them to the widget when you need them.
