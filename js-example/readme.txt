== Description ==

This plugin demonstrates how to use JavaScript in WordPress. 
It demonstrates how to enqueue the javascript and also pass parameters to the 
Javascript File.

To use this file, just copy this plugin in your wp-content/plugins folder and enable it.  
As you navigate to the various pages, you will see the different JS files enqueued and run.

For example, there are different hooks to enqueue on normal pages, admin pages, and login pages.
