This plugin demonstrates how to add a meta box and save various options to a post

== Description ==

just an example on how to add a custom meta box and to save various options to
a post.  Adding a meta box to a post isn't always straightfoward. 
This plugin attempts to explain how to do this, along with providing examples for a dropdown.
