=== Books Plugin ===

Contributors: flacoman91

This is a basic plugin and should only be used for demo and development purposes
to learn how to program.  The goal of this project is to provide a well
documented example for people to learn how to use WordPress and create your own 
plugins.

== Description ==
This plugin is a hacked together example of the various code snippets to show 
how to create a WordPress plugin. 

It will demonstrate how to do various things through a plugin:
* create a custom content type 
* create various form input types and save this information
* custom meta box or custom field.
