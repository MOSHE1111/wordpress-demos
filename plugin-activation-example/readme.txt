== Description ==

This plugin demonstrates how the register activation hook can be used. 
In this example, we create some categories once this plugin is activated.
We also demonstrate how you would activate another plugin when this one is activated.

Please see:

http://codex.wordpress.org/Function_Reference/register_activation_hook
